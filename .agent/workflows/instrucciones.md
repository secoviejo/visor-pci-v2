# Reglas del Proyecto Visor PCI

Como asistente de IA en este proyecto, debes seguir estas reglas en todo momento:

1. **Idioma**: Todas las respuestas, resúmenes de tareas (`task`), planes de implementación y archivos de recorrido (`walkthrough.md`) deben estar redactados estrictamente en **Español**.
2. **Estilo**: Mantén un tono profesional pero cercano, como un colaborador técnico experto.
3. **Documentación**: Cada vez que realices cambios significativos, actualiza o crea un archivo en la carpeta `progresos/` con el resumen de los avances, siguiendo el formato de los archivos existentes (ej. `AVANCES_DD_MM_AAAA.md`).
4. **Calidad de Código**: Asegúrate de que el código sea limpio, documentado y siga las mejores prácticas de Vanilla JS y Node.js.
